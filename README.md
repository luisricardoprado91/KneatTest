### Developed
Windows 10 Home Single Language
# To execute
```
$ cd ..\KneatTest\test
$ mvn install
$ mvn test
```
# To execute
```
$ Run com.kneat.test.Tests as JUnit Test
```

* [Eclipse] - IDE for development
* [JUnit] - Test execution framework

* [Java] - jdk-13.0.2
